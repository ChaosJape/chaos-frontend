**ChaosFrontend** is a frontend application that quickly generates user-defined frontend templates.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Technology stack

- HTML
- CSS
- Javascript
- Vue 3
- vue-router
- element-plus
